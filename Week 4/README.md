# Week 4 – Advanced VAPT Labs

This repository contains the practical labs and documentation for Week 4 of the VAPT training program.

## Labs Covered

1. Advanced Exploitation Techniques
2. API Security Testing
3. Privilege Escalation and Persistence
4. Network Protocol Attacks
5. Mobile Application Penetration Testing
6. Capstone Project – Full VAPT Engagement

## Tools Used

- Kali Linux
- Nmap
- Metasploit Framework
- Burp Suite
- MobSF
- Wireshark
- LinPEAS

## Workflow

1. Reconnaissance using Nmap
2. Vulnerability identification
3. Exploitation using Metasploit
4. Privilege escalation using LinPEAS
5. API testing using Burp Suite
6. Mobile application analysis using MobSF
7. Documentation and reporting
